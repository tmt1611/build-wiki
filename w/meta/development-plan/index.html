K10 Development Plan

This document outlines the roadmap and iteration plan for the writing and development of the K10 world. Please regularly update this file according to the progress of the repo and remove completed tasks.

Phase 5: The Unawakened World (Ongoing)

  • Goals: To significantly deepen the lore and narrative significance of the Unawakened factions, particularly the guilds of Nexus. This phase will shift focus from high-level magical conflicts to the mundane, economic, and social systems that form the bedrock of the K10 world.

Cycle Log

This log tracks the last completed development iteration.

  • Role 2/7 (World Builder): Integrated the new magical-pedagogy.md document more deeply into the three main military files, adding sections that connect a faction's educational system to its military doctrine and personnel.
  • Role 3/7 (Magic System Designer): Completed the planned consistency review. Tightened language in archetypal-builds.md to be less grandiose. Reviewed contested-magic.md, fixed a broken link, and replaced a phantom example with a concrete link to an existing playtest, strengthening the connection between lore and mechanics.
  • Role 4/7 (Reviewer): Executed a comprehensive prose and style audit on the foundational fable, library/folklore-and-superstition.md, improving readability and ensuring adherence to the project's tone.
  • Role 5/7 (Maintainer): Continued the asset management task by converting several inline SVG diagrams into static assets, improving repository hygiene and maintainability.
  • Role 6/7 (Playtester): Logged the "Dream-Sworn and the Mundane Anchor" playtest, successfully validating the lore for the heretic Somnus and the effectiveness of the Lamplighters' mundane tools against high-tier Yesod magic.
  • Role 7/7 (Single Writer): Created the core document library/unawakened-library/unawakened-philosophies.md, detailing the three major belief systems of the non-magical population and integrating them into the wider lore.
  • Role 5/7 (Maintainer): Focused on visual asset creation. Added new SVG generation functions to scripts/generate_thumbnails.py for several high-priority diagrams (oracle_contract_diagram.svg, chokmah_star_moths.svg, unnaming_ritual.svg) to fulfill requests in the visual design goals.
  • Role 1/7 (Lead Writer): Reviewed the previous cycle and updated changelog. Performed a style-guide polish pass on factions-at-war/progenitor-factions.md. Set the plan for the current cycle.
  • Role 5/7 (Maintainer/Asset Creator): Created and integrated several high-priority visual assets for core concepts. This includes diagrams for The Three Pillars, Strategic Doctrines, The Qliphoth Tree, and a new Cosmology Overview diagram. Cleaned up placeholder styles in existing image tags.
  • Role 5/7 (Maintainer/Asset Creator): Created several key visual assets (puzzle_box.svg, somnarium_logo.svg, oracle_contract_diagram.svg, logic_engine.svg, living_tide_fauna.svg) to fulfill requests in the visual design goals and replace placeholders in key documents.
  • Role 5/7 (Maintainer/Asset Creator): Created several high-priority visual assets to define faction aesthetics (architectural_styles_triptych.svg, factional_cuisine_diagram.svg). Also completed the full set of 10 symbolic icons for the Qliphothic Husks and integrated them into the lore.
  • Role 5/7 (Maintainer/Asset Creator): Created and integrated the visual asset for the "Absurdity Offensive" narrative scene, closing a high-priority issue from the reports.

Next Iteration Focus: Reviewer (Role 4/7)

With a new batch of core visual assets and icons integrated, the next cycle should focus on auditing these assets and their context.

  • Task 1 (Primary): Review the newly integrated diagrams (architectural_styles_triptych.svg, factional_cuisine_diagram.svg) and the new Qliphoth icons. Check their containing documents (architectural-styles.md, factional-cuisine.md, the-qliphoth.md) for clarity, descriptive alt text, and proper context.
  • Task 2 (Secondary): The magic-system/sefirot/ documents are some of the oldest in the repository. If time permits after the primary task, begin a prose and style audit on a few of these files (e.g., s1-keter.md, s10-malkuth.md) to bring them in line with the current style-guide.md.

Future Content & Expansion Goals (Next Cycle Preview)

  • Maintainer: The next cycle's Maintainer will focus on addressing any broken links identified by the Reviewer and will continue the ongoing asset management tasks.
  • Asset Creator: Future cycles should continue to work through the prioritized list of visual assets. The next major goal is to create the individual, symbolic icons for the 22 Arcana. This will involve either creating a new script function for generating them or adding them as static assets and then integrating them into the-22-paths.md.

Summary of Completed Work

This section provides a high-level overview of previously completed work. For a detailed history, please see the full changelog.md.

  • Phase 1 - Core Foundation: Established the core documents for the magic system (10 Sefirot, 22 Paths, Corruption, Cycles), the three major factions, and the world's cosmology.
  • Phase 2 - Content Expansion: Built upon the foundation with detailed content, including the full set of Qliphothic Avatars, Sefirotic Realms, and Archetypal Builds. Added RTS profiles, Unawakened perspectives, and narrative content to deepen the world.
  • Phase 3 - System Refinement & Polish: Focused on deepening existing content and ensuring consistency. Key activities included expanding faction military rosters ("Beast-Kin", Unawakened corps), integrating Nexus guild logistics, adding cultural documents (aesthetics, architecture, cuisine, law), refining the magic system with "Invocations," establishing the "War of the Shifting Throne" arc, and validating esoteric archetypes through numerous playtests.
  • Phase 4 (Milestones): Completed the full set of 22 detailed primers for the Arcanum Legions. Completed the detailed mechanical breakdowns for all 22 Paths and all foundational Cycles.
